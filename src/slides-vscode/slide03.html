AZURE INTEGRATION SETUP

STEP 1: AZURE OPENAI RESOURCE

Create Azure OpenAI resource in compliant region. Deploy GPT-4o or GPT-4 Turbo model. Note endpoint URL and deployment name.

STEP 2: GITHUB ENTERPRISE SETTINGS

In GitHub Enterprise → Settings → Copilot → Azure connection. Enter Azure tenant ID, subscription ID, resource group, and deployment.

STEP 3: MANAGED IDENTITY

Configure Azure AD app registration. Grant "Cognitive Services OpenAI User" role. No API keys stored in GitHub.
