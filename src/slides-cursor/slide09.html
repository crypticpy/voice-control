ENTERPRISE RECOMMENDATIONS

KEY MANAGEMENT

Use Azure Key Vault. Distribute keys securely. Rotate regularly. Never commit to git.

COST CONTROL

Monitor via Azure Portal. Set usage alerts. Consider reserved capacity for predictable usage.

TEAM CONSISTENCY

Share .cursor/rules/ via git. Standard patterns across team. Document setup process.

MODEL SELECTION

GPT-4o for most tasks. Test models for your workload. Different models per feature.
