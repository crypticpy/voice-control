CONFIGURATION

Configure
Cursor

SETTINGS PATH

Cursor Settings → Models → API Keys → Azure OpenAI

QUICK ACCESS

Cmd+, (macOS)
Ctrl+, (Windows)
→ Cursor Settings

Always click "Verify" after entering

STEP-BY-STEP SETUP

IN CURSOR SETTINGS → MODELS → API KEYS

1. Click "Add Model" or select "Azure"
2. Enter Base URL: https://yourorg.openai.azure.com/
3. Enter Deployment Name: gpt-4o-deployment
4. Enter API Key: [paste your key]
5. Click "Verify" to test connection
6. Enable for Chat, Cmd+K, and Composer

SUCCESS CHECK

Green checkmark after "Verify" means you're connected to Azure

COMMON ERROR

Deployment name must exactly match Azure — case sensitive!

ENABLE FOR FEATURES

After adding Azure, check the boxes to enable it for: Chat (Cmd+L), Cmd+K (inline), and Composer (Cmd+I). Each can use different models.
