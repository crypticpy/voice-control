IDE INTEGRATION

Wispr Flow + Code Editors

Special features for VS Code, Cursor, and other IDEs

Automatic Code Formatting

Wispr detects when you're in a code file and automatically formats variable names as camelCase, adds proper syntax, and handles special characters.

Language-Aware Dictation

Recognizes file type (.py, .js, .tsx) and adjusts output. Say "string" in Python, get quotes; in TypeScript, get proper typing.

Comment Dictation

Say "comment" before dictating and Wispr adds appropriate comment syntax (// for JS, # for Python, etc.)

CURSOR + WISPR FLOW

Combine voice dictation with Cursor's AI. Dictate prompts to Cursor's composer with your voice for hands-free AI coding.

CONFIGURATION TIPS

Hotkey Setup

Avoid conflicts with IDE shortcuts. Use Cmd/Ctrl+Shift+D or a dedicated key like Caps Lock.

Enable IDE Mode

Settings > Applications > Enable "Developer Mode" for enhanced code formatting.

Custom Vocabulary

Add your project's function names, variables, and technical terms for better recognition.

WORKFLOW EXAMPLE

1. Open Cursor/VS Code
2. Position cursor where you want code
3. Hold hotkey and describe function
4. Release to insert formatted code

Works with any text-based IDE: JetBrains, Vim, Emacs, etc.
