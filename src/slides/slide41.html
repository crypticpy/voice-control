CONFIGURATION

Azure OpenAI Setup

1

Request Access

Apply at aka.ms/oai/access. Approval typically 1-2 business days.

2

Create Resource

Azure Portal > Create Azure OpenAI resource in your subscription.

3

Deploy Model

Azure AI Studio > Deploy GPT-4 or GPT-4o model.

Environment Variables

AZURE_OPENAI_ENDPOINT=https://your-resource.openai.azure.com

AZURE_OPENAI_API_KEY=your-api-key

AZURE_OPENAI_DEPLOYMENT=your-deployment-name

AZURE_OPENAI_API_VERSION=2024-02-15-preview

These credentials work with Continue.dev, LangChain, and other AI development tools
