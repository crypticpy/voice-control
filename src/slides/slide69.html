WHAT CUSTOM MODES DO

Process your dictation through AI with specific instructions. Create specialized workflows for different tasks.

FREE TIER

3 custom modes included

PRO TIER

Unlimited custom modes

Local AI processing keeps your data on-device

STEP-BY-STEP CREATION

1. Access Mode Settings

Click menu bar icon → Modes → Create New Custom Mode

2. Name Your Mode

Choose descriptive name (e.g., "Email Writer", "Code Commenter")

3. Write AI Instructions

Define how AI should process your dictation (see next slide)

4. Select Voice + Language Models

Choose local models for privacy, cloud for speed/accuracy

5. Configure Context (Optional)

Enable app context, clipboard context, selected text context

6. Add Examples (Recommended)

2-3 input/output examples dramatically improve accuracy

7. Assign Hotkey (Optional)

Quick-switch to this mode with a keyboard shortcut
