AUTOMATION

Hooks System

Run scripts at key points in Claude's workflow

PreToolUse

Run before Claude executes a tool (read, write, bash)

PostToolUse

Run after a tool completes (log actions, format code)

Notification

Handle Claude notifications (Slack, email, etc.)

Use case: Auto-format code before commits, log all file writes

EXAMPLE HOOK

// settings.json

{

"hooks": {

"PostToolUse": [{

"matcher": "Write",

"hooks": [{

"type": "command",

"command": "prettier --write"

}]

}]

}

}

Auto-format files after Claude writes them
