ENTERPRISE COMPLIANCE

VS Code with Azure OpenAI

Configuring Continue.dev for government-compliant AI

CONFIG.JSON

// ~/.continue/config.json

{

"models": [{

"provider": "azure",

"model": "gpt-4",

"apiBase": "https://your-

resource.openai.azure.com",

"deployment": "your-deploy",

"apiVersion": "2024-02-15"

}]

}

Why This Matters

Your code stays in your Azure tenant. FedRAMP compliant. Data not used for training.

Requirements

1. Azure OpenAI access approved
2. GPT-4 deployment created
3. API key from Azure Portal
4. Continue.dev extension installed

Environment Variable

AZURE_OPENAI_API_KEY=your-key

Note: Keep API keys secure - use env vars, not config files
