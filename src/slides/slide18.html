INSTALLATION

Wispr Flow
Setup Guide

Quick setup on any platform

1

Visit wisprflow.ai

Download for your platform (Mac/Windows/iOS)

2

Install the Application

Mac: drag to Applications | Windows: run installer

3

Create Account or Sign In

Free tier available, or connect existing account

4

Grant Microphone Permission

Required for voice input functionality

5

Enable Privacy Mode (IMPORTANT)

Settings > Data and Privacy > Enable Privacy Mode

Note: Privacy Mode = zero data retention (audio not stored). Audio still processed in cloud.
