WARNINGS

Common Mistakes to Avoid

NEVER DO THIS

Paste PHI into public AI tools

ChatGPT, Claude.ai free tier, etc.

Include real patient data in prompts

Even for "testing" purposes

Use cloud dictation for PHI

Default OS dictation sends to cloud

Share API keys or credentials

In code, prompts, or conversations

Assume "private" means compliant

Verify with IT/compliance first

DO THIS INSTEAD

Use approved enterprise tools

Azure OpenAI, Azure AI Foundry

Use synthetic or anonymized data

For testing and development

Use local voice transcription

SuperWhisper, WhisperFlow

Use environment variables

Never hardcode credentials

Verify compliance before use

Check with IT security team

REMEMBER

"If you wouldn't print it and leave it on a public printer, don't paste it into a public AI."

The convenience of AI tools doesn't change our responsibility to protect sensitive data.

When in doubt, ask IT Security
