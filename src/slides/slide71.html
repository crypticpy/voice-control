ADVANCED FEATURES

Context Awareness & Mode Switching

THREE CONTEXT TYPES

APPLICATION CONTEXT

Knows which app is active (VS Code, Teams, Chrome, etc.)

Captured: After voice processing, before AI

SELECTED TEXT CONTEXT

Passes highlighted text to AI for reference or transformation

Captured: Exactly when recording starts

CLIPBOARD CONTEXT

Uses recently copied content as reference material

Captured: 3 sec before OR during recording

SUPER MODE NOTE

Super Mode enables ALL three by default. Custom modes let you choose which to enable.

FOUR WAYS TO SWITCH MODES

1. KEYBOARD SHORTCUTS

Assign unique shortcuts per mode
Configure in Settings → Keyboard

2. MENU BAR / SYSTEM TRAY

Click SuperWhisper icon → Select mode
Visual indicator shows current mode

3. AUTO-ACTIVATION RULES

Auto-switch when specific apps are active
e.g., VS Code → Coding Mode

4. DEEP LINKS

superwhisper://mode/[name]
Integrate with Alfred, Raycast, scripts

AUTO-ACTIVATION SETUP

EXAMPLE RULES

VS Code → "Coding Mode"
Teams → "Chat Mode"
Outlook → "Email Mode"
Chrome → "Browse Mode"
Slack → "Slack Mode"

HOW TO SET UP

1. Edit your custom mode
2. Go to Advanced Settings
3. Add Auto-Activation Rule
4. Select app or enter URL pattern

POWER USER TIP

Combine auto-activation with context awareness for fully adaptive dictation workflows.
