KEY CAPABILITIES

Wispr Flow Features

Context Awareness

Automatically detects what app you're using. Formats output for code in VS Code, prose in Word, casual in Slack.

Command Mode

Say "Hey Wispr" to trigger commands. Edit, format, translate, or perform actions on selected text.

Flow State Mode

Continuous dictation without holding keys. Start speaking when ready, pause naturally between sentences.

Smart Formatting

Auto-capitalizes sentences, adds punctuation intelligently, formats code blocks, and handles special characters.

Multi-Language Support

Supports 30+ languages with automatic detection. Switch languages mid-dictation seamlessly.

Correction Mode

Say "correct that" or "change X to Y" to make inline edits without switching to keyboard.

CODING CONTEXT EXAMPLE

Say: "create a function called calculate total that takes price and quantity"
Output: function calculateTotal(price, quantity)

EMAIL CONTEXT EXAMPLE

Say: "hi john comma thanks for the meeting period looking forward to next steps"
Output: Hi John, Thanks for the meeting. Looking forward to next steps.
