ESSENTIAL COMMANDS

Claude Code Key Commands

Session Management

/clear

Reset conversation history. Start fresh without restarting Claude.

/compact

Summarize context to reduce token usage while preserving key info.

/resume

Continue a previous session where you left off.

Information

/help

Display all available commands and their descriptions.

/cost

Show token usage and estimated cost for current session.

/status

View current session info, model, and context window usage.

Project Setup

/init

Create a CLAUDE.md file with project context and conventions.

/config

View or modify Claude Code settings and preferences.

/add-dir

Add additional directories to the current context.
