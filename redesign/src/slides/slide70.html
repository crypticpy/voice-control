CUSTOM MODE PROMPTS

Writing Effective AI Instructions

PROMPT STRUCTURE

<role>

You are a professional email editor

</role>

<instructions>

Format the User Message into a
professional email with proper
greeting and signature

</instructions>

<requirements>

Keep tone professional
Use clear paragraphs
Include appropriate sign-off

</requirements>

XML TAGS NOTE

XML tags improve clarity with advanced models. Some local models may struggle with them - test your results.

KEY ELEMENTS

1. ROLE STATEMENT

Define who the AI is. Sets the foundation for all processing.

2. CLEAR INSTRUCTIONS

Explicit actions to take with the dictated text. Be specific.

3. REQUIREMENTS LIST

Bullet points of constraints and style guidelines.

4. EXAMPLES (2-3)

Show input → output pairs. Dramatically improves accuracy.

CONTEXT REFERENCES

Use "User Message" for dictated text,
"Application Context" for current app,
"Clipboard Context" for copied text

BEST PRACTICES

DO

✓ Be explicit and specific
✓ Include 2-3 examples
✓ Use modular sections
✓ Test with your voice
✓ Iterate and refine

AVOID

✗ Vague instructions
✗ Conflicting directives
✗ Too many requirements
✗ No examples
✗ Untested prompts

TELL AI IT'S DICTATED

"The User Message contains dictated speech. Please format..."
