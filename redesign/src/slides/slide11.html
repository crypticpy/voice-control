INSTALLATION GUIDE

NEW

SuperWhisper for Windows

Step-by-step installation

1

Download Installer

Visit superwhisper.com/windows and download the .exe installer

2

Run Installer

Double-click installer, accept UAC prompt, follow wizard

3

Configure Microphone Access

Settings > Privacy > Microphone > Allow SuperWhisper

4

Download AI Model

Select model on first launch (Medium recommended). ~1-2GB download.

5

Test with Hotkey

Press Ctrl+Shift+Space (default) to start recording. Press again to transcribe.

GPU Recommended: NVIDIA GPU with CUDA significantly improves speed.

WINDOWS REQUIREMENTS

Minimum Specs

Windows 10 (64-bit) or Windows 11
8GB RAM
4GB free storage
Any modern CPU

Recommended Specs

NVIDIA RTX 2060 or better
16GB RAM
CUDA 11.0+ drivers installed
SSD storage

FIRST RUN CHECKLIST

1. Allow through Windows Firewall
2. Set to run at startup (optional)
3. Configure system tray icon
4. Test microphone levels

Tip: Pin SuperWhisper to taskbar for quick access
