INSTALLATION GUIDE

SuperWhisper for Mac

Step-by-step installation

1

Download SuperWhisper

Mac App Store or superwhisper.com direct download

2

Install the Application

Drag to Applications folder or click Install in App Store

3

Grant Microphone Access

System Settings > Privacy > Microphone > Enable SuperWhisper

4

Complete Onboarding

Follow guided setup to test microphone and select voice model

5

Test Recording

Use hotkey to record, release to transcribe. Text appears at cursor.

DOWNLOAD OPTIONS

Option B: Direct Download
