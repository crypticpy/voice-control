CONTEXT SETTINGS

SAMPLE AI INSTRUCTIONS

<role>

You prepare task specifications for AI coding agents

</role>

<instructions>

Convert the User Message into a structured task.
Include: objective, scope, constraints, success criteria.

</instructions>

<requirements>

Clear, actionable objective
Reference files from Selected Text if provided
Include Clipboard context if relevant
Specify what NOT to change if mentioned

</requirements>

RAW SPEECH

"okay so there's a bug in the user dashboard where the charts aren't loading and I think it's something to do with the API call timing and we need to fix it but don't touch the CSS because design just approved that"

STRUCTURED HANDOFF

Objective: Fix chart loading bug in user dashboard
Likely cause: API call timing issue
Scope: JS/API logic only
Constraint: Do NOT modify CSS
Success: Charts load correctly

WHY THIS MATTERS

AI agents work better with structured tasks than rambling instructions.

WORKFLOW

Select relevant code → Dictate task → Paste to Claude Code → Agent executes
