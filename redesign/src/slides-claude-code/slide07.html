AUTO-DISCOVERY

Skills

THE KEY DIFFERENCE

Unlike slash commands, skills activate automatically when task context matches.

HOW THEY WORK

1. You give a task
2. Claude reviews skill descriptions
3. Matching skills auto-load
4. Instructions apply automatically

No manual invocation needed

SKILLS VS SLASH COMMANDS

SLASH COMMANDS

Manual invocation
You type /command
Explicit workflow triggers
Best for: specific actions

SKILLS

Automatic activation
Context-based matching
Always-on capabilities
Best for: domain expertise

SKILL FILE STRUCTURE

.claude/skills/react-patterns.md

# Description (for matching)
"Use for React component development"

# Instructions (loaded when matched)
"Follow these patterns..."

USE CASE

Domain-specific patterns, coding standards, framework guidelines
