CORE FEATURE

Slash
Commands

WHAT THEY ARE

Manual commands you invoke by typing /command in the prompt.

HOW TO ACCESS

Type / to see all available commands
Tab to autocomplete
Arrow keys to navigate

Foundation for customization

BUILT-IN COMMANDS

/plan — Enter planning mode
/model — Switch models
/compact — Summarize context
/clear — Reset conversation

/help — Show all commands
/config — Open settings
/review — Code review mode
/cost — Show token usage

CUSTOM COMMANDS

CREATE YOUR OWN

# Add markdown files to:
.claude/commands/my-command.md

# Then invoke with:
/project:my-command

TEAM SHARING

Commit .claude/commands/ to git — everyone gets the same commands

USE CASE

Standard prompts for PRs, commits, code review templates
