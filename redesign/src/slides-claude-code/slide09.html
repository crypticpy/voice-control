Deterministic code that runs at specific points in Claude's execution pipeline.

WHY USE HOOKS

Transparency & auditability
Security enforcement
Cost tracking
Policy compliance

Enterprise-grade control

HOOK TYPES

PreToolUse

Before tool execution. Can block calls, validate permissions.

PostToolUse

After tool execution. Log results, inject metrics.

PermissionRequest

When permission dialog shown. Auto-approve or deny.

AgentCompletion

When agent finishes. Clean up resources, validate outputs.

Block sensitive file access, enforce allowed directories
