AZURE OPENAI

Getting Azure
Credentials

WHAT YOU NEED

Three values from Azure Portal to connect Cursor to your organization's models.

REQUIRED VALUES

1. Base URL (endpoint)
2. Deployment name
3. API key

portal.azure.com

FINDING YOUR AZURE CREDENTIALS

1

Navigate to Azure OpenAI Resource

Azure Portal → All Resources → Your Azure OpenAI Service

2

Copy Endpoint & Key

Keys and Endpoint → Copy KEY 1 and Endpoint URL

3

Get Deployment Name

Model deployments → Copy exact deployment name (e.g., gpt-4o-deployment)

EXAMPLE VALUES

Base URL: https://myorg.openai.azure.com/
Deployment: gpt-4o-deployment
API Key: abc123...
