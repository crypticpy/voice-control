WHAT IS CURSOR

AI-native code editor built on VS Code. Same interface, AI superpowers built in.

REQUIREMENTS

Windows 10+ / macOS 10.15+ / Linux
4GB RAM minimum
Internet connection
Azure subscription (for BYOK)

Downloads at cursor.com

INSTALLATION BY PLATFORM

WINDOWS

Download .exe installer
Run setup wizard
Follow prompts

MACOS

Download .dmg file
Drag to Applications
Allow in Security prefs

LINUX

Download .AppImage
chmod +x cursor.AppImage
Run directly
