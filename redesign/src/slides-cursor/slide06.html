CONNECTING FOUNDRY TO CURSOR

OPENAI-COMPATIBLE MODELS

Models deployed via Azure OpenAI Service (GPT-4, GPT-4o) work with the standard Azure configuration. Use the same setup as Azure OpenAI.

OTHER FOUNDRY MODELS

Claude, Llama, and Mistral deployed via Foundry require checking the specific endpoint format. Some may need OpenAI-compatible API wrapper.

FOUNDRY MODEL ENDPOINT FORMAT

# Check your Foundry deployment for:
Endpoint URL: https://<project>.<region>.inference.ml.azure.com/
API Key: [from Foundry portal]
