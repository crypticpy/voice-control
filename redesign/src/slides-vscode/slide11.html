OLLAMA + VS CODE SETUP

1. INSTALL OLLAMA

$ curl -fsSL https://ollama.com/install.sh | sh

or download from ollama.com

2. PULL A MODEL

$ ollama pull codellama:13b

or: deepseek-coder, qwen2.5-coder, starcoder2

3. CONFIGURE EXTENSION

In Continue/Cline/CodeGPT settings:
Provider: Ollama
Base URL: http://localhost:11434
Model: codellama:13b

RECOMMENDED MODELS

codellama:13b (balanced)
deepseek-coder:6.7b (fast)
qwen2.5-coder:14b (capable)

HARDWARE NEEDS

7B: 8GB VRAM
13B: 16GB VRAM
33B+: 24GB+ VRAM
