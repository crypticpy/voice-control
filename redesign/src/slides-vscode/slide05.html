AI TOOLKIT CAPABILITIES

MODEL CATALOG

Browse Hugging Face models optimized for local inference. One-click download.

PLAYGROUND

Test models interactively. Compare outputs. Tune parameters before integration.

FINE-TUNING

LoRA/QLoRA workflows for customizing models on your data. Export ONNX.

HARDWARE SUPPORT

NVIDIA CUDA, AMD ROCm, Intel NPU, Apple Silicon acceleration.

QUICK START

1. Install "AI Toolkit" from VS Code marketplace
2. Open AI Toolkit sidebar → Model Catalog
3. Download a model (e.g., Phi-3-mini)
4. Open Playground → Test inference

NOTE

AI Toolkit is for model experimentation and fine-tuning — not inline code completion like Copilot.
